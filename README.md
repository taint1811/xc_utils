## Introduction

Base build tool for HTML/CSS

## Require

- Nodejs
- npm-run-all
- browser-sync
- nodemon
- sync-glob
- copyfiles
- rimraf
- pug
- minimist

## Command line

Run serve with hot reload at localhost:3000.

```bash
$ npm run serve
```

Build for production with minification

```bash
$ npm run build
```
